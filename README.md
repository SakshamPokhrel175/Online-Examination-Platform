# Online Examination Platform

A hybrid secure online examination platform with exam and quiz (pool) modes.

## Features
- AI-based proctoring (webcam, tab detection)
- Role-based login (Admin, Teacher, Student)
- Exam mode (secure)
- Pool mode (practice quizzes)
- JWT Authentication

## Tech Stack
- Frontend: Angular
- Backend: Spring Boot
- Database: MySQL

## Project Structure
- backend_springboot/
- frontend_angular/
